# Vshapes React

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) installed

### Installation

1. Clone the repo
   git clone https://github.com/YOUR_USERNAME/vshapes-react.git

2. Go into the folder
   cd vshapes-react

3. Install dependencies
   npm install

4. Run the app
   npm run dev

5. Open your browser at
   http://localhost:5173
